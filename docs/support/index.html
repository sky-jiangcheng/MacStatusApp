Frequently Asked Questions

How do I refresh the running app list?
Use the refresh button in the toolbar. MacStatus also updates the list automatically.

How do I quickly hide MacStatus?
Use the Hide command or its keyboard shortcut from the app menu.

Does MacStatus collect personal data?
No. Running app information is processed locally and is never uploaded.

System Requirements

MacStatus requires macOS 14.0 or later.
